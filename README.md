# 🚀 TrustGrid AI  
### 🏙️ Digital Transparency for Public Infrastructure in Bharat  

**TrustGrid AI** is an AI-powered civic transparency platform developed by **Team CivicSense**, designed to bridge the gap between citizens and government by enabling **real-time monitoring, accountability, and trust measurement** in public infrastructure projects.

---

## 📌 Problem Statement  

Public infrastructure projects in India often suffer from:

- ❌ Lack of transparency in budgets, timelines, and contractors  
- 📉 Ignored citizen complaints  
- 🕒 Delays and inefficiencies  
- 📄 Manual reporting systems (paper/phone-based)  
- ⚠️ No accountability tracking  
- 📊 No measurable trust metrics  

These issues lead to a **significant loss of public trust** and inefficient governance.

---

## 💡 Solution  

**TrustGrid AI (CivicSense Platform)** provides a unified system connecting:

👤 Citizens + 🏛️ Government + 🏗️ Infrastructure  

### Key Modules:

- 📊 **Citizen Dashboard**
  - Live project tracking  
  - Trust scores  
  - Interactive maps  

- 📸 **AI-Based Issue Reporting**
  - Upload image → AI auto-classifies issue  
  - Categories: potholes, pipeline bursts, safety hazards  

- 🛠️ **Admin Portal**
  - Manage infrastructure projects  
  - Respond to complaints  
  - Track issue resolution  

---

## 🌟 USP & Key Features  

- 🧠 **TrustGrid Score (0–100)**  
  - Based on progress, delays, budget efficiency, and citizen reports  

- 🤖 **AI Image Classifier**  
  - Automatically classifies uploaded issue images  

- 🗺️ **Live Google Maps Integration**  
  - Real-time project markers with health status  

- 🎤 **Voice-Based Reporting**  

- 📊 **Advanced Data Visualization**  
  - Bar, Line, Doughnut, Radar, Polar Area, Horizontal Bar charts  

- 👥 **Dual Portal System**  
  - Citizen + Government interfaces

---

## 🔄 Workflow  

1. 🔐 User logs in via Firebase  
2. 📊 Dashboard displays real-time KPIs  
3. 🗺️ Map shows live projects and issues  
4. 📸 User submits issue → AI classifies it  
5. 🏛️ Admin reviews and resolves issues  
6. 📈 Analytics dashboard visualizes insights
